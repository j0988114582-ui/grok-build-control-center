## Public Issues

Use the repository Issue forms for reproducible bugs, focused feature requests, and Windows beta feedback. Search existing Issues before opening a new one.

This community project does not provide guaranteed response times. Reports with a clear app version, Windows version, Grok CLI version, reproduction steps, and sanitized evidence are easier to triage.

## Security

Do not publish vulnerabilities, credentials, authentication data, destructive-operation bypasses, or private project content in an Issue. Use GitHub Security Advisories for private disclosure.

## xAI and account support

This project is unofficial and is not affiliated with or endorsed by xAI. Questions about Grok subscriptions, account access, billing, authentication service availability, or official CLI policy must be handled through xAI's official support channels.
