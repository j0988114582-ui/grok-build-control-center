  - 目前 active 或在 Team slots

永不 suggested-cleanup：pinned
永不自動刪

## 8. 手機 QR 遙控 — 外網 + 安全（Codex 修訂版）

### 8.1 定位

- 類似 Codex 掃碼遠端核准／下指令，但是 **本機控制台網頁 + 出站隧道**。  
- **App 內授權永遠必須**；Cloudflare Access 是加防，不是替代。

### 8.2 隧道策略

| 模式 | 定位 | 0.8.0 |
| --- | --- | --- |
| **Quick Tunnel** | **Experimental / best-effort**；每次啟動風險確認；無 SLA；**不支援 SSE**；in-flight 約 200 | 一鍵試用 |
| **Named Tunnel + Access** | 重複使用推薦；使用者自備；Access JWT 建議在 origin 驗證 | 設定項支援 |
| 任意 public base URL | **砍出 MVP**（X9）；或僅「HTTPS origin pin + nonce health」進階後做 | 不做預設 |

**本機永遠 `127.0.0.1` only。**  
QR **僅在**「nonce health check 經隧道打回本進程成功」後顯示。

**事件推送：** Quick 模式用 **有界 polling 合併 snapshot**（status+permissions+tail），**不用 SSE**。

### 8.3 MVP 能力

| 能力 | |
| --- | --- |
| 配對 + PIN + session cookie | ✓ |
| status / 簡化 session 列表（**預設不含 cwd**） | ✓ |
| prompt / cancel（session **伺服端選定**，不信任 client sessionId） | ✓ |
| permission.respond（**物件級**綁定） | ✓ 另需桌面開關「允許手機核准權限」**預設關** |
| transcript.tail 精簡 | ✓ 有上限 |
| 切斷／撤銷 | ✓ |
| YOLO 遠端切 | ✗ |
| 遠端刪 session／設定／附件上傳 | ✗ |

### 8.4 安全 Must

#### L0 網路

| ID | 要求 |
| --- | --- |
| R-SEC-0 | 僅 `127.0.0.1`；隧道指 localhost |
| R-SEC-0b | 禁止 `0.0.0.0` 當外網方案 |
| R-SEC-0c | 關遙控：殺 tunnel + 關 HTTP + 作廢全部 token/session |
| R-SEC-0d | **精確 Host + Origin allowlist**；**無 CORS**；body/header/時間限制；fail-closed |
| R-SEC-0e | QR 前：**nonce health** 證明公網 URL 回到本進程 |

#### L1 配對

| ID | 要求 |
| --- | --- |
| R-SEC-1 | QR：`https://<host>/#/pair?t=<pairingSecret>`；secret ≥128-bit CSPRNG |
| R-SEC-1b | SPA **讀 fragment → `history.replaceState` 立刻剝除 → JSON POST `/api/pair`**；**永不**進 query／log／DOM 明文／referrer |
| R-SEC-2 | Pairing TTL **3 分**；倒數；**過期只關閉配對，不自動無限換碼**；桌面按一下才重新產生 |
| R-SEC-2b | 單設備成功配對後 **立刻關閉 pairing** |
| R-SEC-3 | Session token 不透明隨機；**伺服端只存 hash（記憶體）**；`Set-Cookie: … HttpOnly; Secure; SameSite=Strict; Path=/api; Host-only`；**不**回 JSON、不進 JS storage |
| R-SEC-4 | Idle **30 分** + 絕對 **4 小時**；重啟／關 Remote／隧道失敗／登出／**權限模式變更** → 全撤銷 |
| R-SEC-5 | **每次啟用 Remote 桌面顯示新 PIN**（6–8 位）；只存 **記憶體 salted slow hash**；**不**寫 electron-store 明文；5 次失敗 **作廢該代 pairing** 並需桌面再產；計數含 pairing 代次（防換 IP） |
| R-SEC-6 | **不把 UA 當持有因素**；預設 **1 活躍裝置**；可撤銷 |

#### L2 傳輸

| ID | 要求 |
| --- | --- |
| R-SEC-7 | 突變：`Content-Type: application/json` + `X-Grok-Remote` + cookie session |
| R-SEC-8 | CSP 靜態 app：`default-src 'self'; script-src 'self'; connect-src 'self'; object-src 'none'; base-uri 'none'; frame-ancestors 'none'`；`Referrer-Policy: no-referrer`；`X-Content-Type-Options: nosniff`；`Cache-Control: no-store` |
| R-SEC-9 | **DTO 最小化**：session 列表預設無 cwd；tail 排除 thought／raw tool input／完整 tool output／圖／大檔；條數與位元組 cap；權限卡僅 **typed 摘要**（工具名、操作類、短路徑／命令摘要、風險、選項、到期）；桌面可「看完整」 |
| R-SEC-10 | Rate limit + **每 session 同時僅 1 個 in-flight prompt**；合併 snapshot polling + backoff |

#### L3 授權

| ID | 要求 |
| --- | --- |
| R-SEC-11 | allowlist 方法 only |
| R-SEC-12 | 禁止 settings／delete／export／YOLO／fs／install |
| R-SEC-13 | 遠端只收文字；長度 cap |
| R-SEC-14 | `permission.respond` 必須匹配伺服端 pending **`{requestId, sessionId, allowedOptionIds, expiresAt}`**，單次消費；錯 session/option → 拒 |
| R-SEC-14b | **Remote ↔ YOLO 互斥**（啟動任一方時檢查另一方） |
| R-SEC-14c | 「允許手機核准權限」**預設關**，需桌面另開 |

#### L4 審計 UX

| ID | 要求 |
| --- | --- |
| R-SEC-15 | 橫幅狀態機：`starting` / `url_verified` / `pairable` / `paired` / `tunnel_failed` |
| R-SEC-16 | 遠端動作桌面標記「來自手機遙控」 |
| R-SEC-17 | **每次**開 Quick：**風險確認**（含供應商可處理 HTTP 內容，見 §8.6） |
| R-SEC-18 | 總開關預設關；不開機自啟 |
| R-SEC-19 | 結構化 log、有界、**redact** URL/secret/cookie/PIN/Authorization/body；cloudflared **非 debug** |

#### L5 供應鏈

| ID | 要求 |
| --- | --- |
| R-SEC-20 | cloudflared **釘版本 + checksum**；驗證後才執行；關子程序 auto-update；**不** `shell:true`；named token OS 安全存放；notices |
| R-SEC-21 | 不把 tunnel URL 送 telemetry |

### 8.5 隱私文案（必須正確）

- TLS 保護手機↔Cloudflare、cloudflared↔edge 等鏈路。  
- **Cloudflare 在邊緣終止訪客 TLS，技術上可處理 HTTP 內容**（提示、權限摘要、transcript 片段）後再轉入隧道。  
- **不可**寫成「供應商只見 SNI／流量形狀、內容全程端到端加密對供應商不可見」。  
- 用完切斷；勿分享 QR 螢幕。
