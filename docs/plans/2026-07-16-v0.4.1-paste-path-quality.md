## 範圍

### M1 品質（P2）

1. Session **載入中** disable 權限模式下拉（需 React state，不能只靠 ref）
2. YOLO 確認防連點（busy 鎖 / disable 確認鈕）
3. busy 文案分流：`running` vs `lifecycleBusy`
4. checkbox 不嵌在 `<button>` 內（合法 HTML）

### M2 貼圖 → 路徑

1. Main IPC：`paste:save-image`（或同等）寫入 `%TEMP%\grok-build-gui-paste\paste-<timestamp>.<ext>`
2. mime → 副檔名（png/jpeg/webp/gif）；大小上限沿用既有 20MB
3. preload/bridge 暴露 API
4. `App.tsx` paste：
   - `promptCapabilities.image === true` → 維持 image block（未來兼容）
   - 否則 → 落檔 → **只**把絕對路徑插入 draft（可換行附加）
5. 小 UI：
   - notice：`已改以本機路徑附上（ACP 目前不支援內嵌圖片）`
   - 可選：composer 附近顯示最近插入路徑的 chip（可移除文字即可，不必 image block）
6. **不做**：自動「請讀取此圖」提示句；真 image/audio ACP block；大視覺改版
