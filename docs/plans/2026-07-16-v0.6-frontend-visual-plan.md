## 3. 三套風格方向（圖稿必須各出 1 張）

共用畫面結構（三張都要畫同一資訊架構，方便比較）：

```
[ Titlebar: 品牌 | Context pill | 額度環 | 權限 | 連線 ]
[ 側欄: 新Session | 搜尋 | 已釘選 | 專案分組 | 對話列 ]
[ 主區: 標題 | 工具列 | 對話串流 ]
[ 底: 狀態「已排入…」| 路徑chip+縮圖 | 輸入框 | 插話 | 立刻改做 | 停止 ]
```

### 風格 A — **銀河座艙進化（Current+）**

- 延續現有深藍 + 金 accent、星空、紙感 transcript  
- 強化層級與指揮列，**不大變品牌 DNA**  
- 適合：想「更精緻的現在」，風險最低  

### 風格 B — **暖紙工作台（Notebook Warm）**

- 米紙／牛皮紙感主區，側欄仍深或半深  
- 手作筆記氣質（對齊 Jackie 講義品味），少霓虹  
- 適合：長時間閱讀、溫暖不科幻  

### 風格 C — **清晰專業（Clarity Pro）**

- 高對比、扁平、間距大方、少裝飾  
- 金 accent 收斂成小點綴；字級偏大  
- 適合：小白一眼看懂、投影演示

每張：**16:9 桌面 App UI mock**，繁中標籤可用短英文示意避免亂字，但版面需清楚。

選型後再：HTML/CSS 實作 + `web-design-shots` 截圖對照。

3. 空狀態 + 媒體縮圖  
4. 字級預設檔 + reduced-motion  
5. 視覺回歸截圖
